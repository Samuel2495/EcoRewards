# EcoRewards - Sustainable Waste Management Platform

EcoRewards is a rewards-based recycling platform designed to encourage community participation in waste segregation and responsible disposal.

## 🚀 Quick Start

### Installation
```bash
npm install
```

### Run Locally
```bash
npm run dev
```

## 🛠 Features

### For Users
- **Eco-Tasks**: Complete sustainable activities to earn points and credits.
- **Nearest Hub Discovery**: Locate the closest disposal center based on your current GPS location.
- **Bulk Recycle**: Schedule pickups for large quantities of recyclables at your doorstep.
- **Rewards Store**: Redeem your hard-earned points for vouchers and donations.

### For Admins
- **Verification Queue**: Review and approve eco-tasks completed by users.
- **Pickup Management**: View and process bulk recycling requests for your specific hub.
- **Hub QR**: Generate a unique QR code for your hub to facilitate user check-ins.

## 📁 Project Structure

- `src/context/`: Core business logic (Auth and Data state management).
- `src/pages/`: Main application views (Dashboard, Hubs, Admin).
- `src/components/`: Reusable UI elements and layout wrappers.
